Things to Learn

May 22nd, 2009

One of my friends recently made a post about the things he need to learn to get a job. Here is the list edited a little and checked for what I have accomplished so far:

  • Basic DNS Concepts
  • Strong Networking and Internet Fundamentals
  • Basic know-how about protocols like HTTP, FTP, SCP, ssh, etc
  • Linux server administration skills
  • Good Linux CLI skills
  • Windows 2000/2003 server administration skills
  • IIS administration and basic ASP skills
  • Database Servers Administration (PostgreSQL/MySQL/MS-SQL) and General SQL usage skills
  • RAID, File Systems (ext{234}/NTFS)
  • Firewalls (Iptables)
  • Apache, PHP, CGI, WSGI
  • Basic knowledge of Postfix/Sendmail/qmail MTA’s, and good knowledge of how E-Mail works
  • Good knowledge about editors (vi, pico, joe, nano, emac) in Linux
  • Regular expressions basics and one liners in sed, awk, Perl etc
